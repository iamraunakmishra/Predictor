# Options Analytics & Risk Engine
A Python-based quantitative tool for options pricing and risk simulation.

## Features
- **Black-Scholes Engine:** Calculates fair value and Greeks (Delta, Gamma, Vega, Theta).
- **Implied Volatility Solver:** Uses the Newton-Raphson method to extract market sentiment.
- **Monte Carlo Simulator:** Runs 10,000 simulations to visualize price probability distributions.

## Tech Stack
- Python (NumPy, SciPy, Matplotlib)
